KotaLab E-commerce store.

A simple  mini e-commerce web app built HTML,CSS
and Vanilla JavaScript.
User can browse kota meals,adjust quantities, add items to cart and
view their order on a dedicated  cart page.Cart data is saved using  localstorage.
so it stays even after refreshing.

Features:
● Product grid with images,names and prices.
● Quality control(+/-)
● add to cart with Validation(No zero quantity)
● Cart page showing items
,totals and delete options.
● Data persistence using localstorage.
● fully responsive layout (flexbox and grid)
